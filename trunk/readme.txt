Clinic review form and patient lead form for a clinic in dentistfind.com.

== Description ==

This plugins allows to fetch reviews and posts reviews and take patient appointment for a clinic in http://dentistfind.com

This plugin requires gravity form plugin installed for patient lead form to be working


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `dental-marketing-system` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to setting page(e.g. http://example.com/wp-admin/options-general.php?page=dentistfind_profile_review) to set api key and other settings
1. Place short code `[dentistfind-profile-review]` for clinic review listing and review form
1. For patient lead form there is a form setting option for api key and clinic permalink
